Rediseño Web Museo

Descripción

Desarrollé una propuesta de rediseño para la página web del Museo de las Culturas con el objetivo de hacerla más atractiva, moderna y accesible para los visitantes. La nueva interfaz busca resaltar la riqueza cultural del museo mediante una estética visual más dinámica, una estructura de navegación clara y una presentación más envolvente de las exposiciones. Desarrollé el prototipo en Figma, donde estructuré el diseño, implementé transiciones para mejorar la experiencia de usuario, aportando fluidez durante la navegación.
